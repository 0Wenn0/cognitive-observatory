## ✦ Decisiones con ciencia

Cada valor en el sistema de diseño tiene evidencia detrás:

| Decisión | Evidencia científica |
|---|---|
| Fondo `#080b12` — azul profundo, no negro | Mehta & Zhu (2009, *Science*): azul activa pensamiento contemplativo |
| Animaciones en ciclos de 3.5s | Berger & Bhagavan (1984): sincronía con ritmo respiratorio humano |
| Texto en el tercio izquierdo | Rayner (1998): primera fijación ocular natural en lectores occidentales |
| Destello `#C8A84B` en el momento pre-revelación | Zeki (1993): detectado en <200ms en fondos oscuros |
| Manifiesto en español, coda en inglés | Cummins (2000): L1 tiene mayor precisión conceptual que L2 |
| Tipografía triple (Serif / Sans / Mono) | Pettersson (2002): cada fuente activa un modo cognitivo distinto |
